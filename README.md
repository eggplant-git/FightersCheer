# 🔥 FIGHTERS 応援歌全曲集

北海道日本ハムファイターズの応援歌をスマホで見やすく一覧できるウェブページです。

## 📋 収録内容

- **選手別応援歌**（23選手）
- **共用ヒッティングマーチ**（メインテーマ1〜3・投手テーマ・Go To Champion）
- **チャンステーマ**（ジンギスカン・チキチキバンバン・北の国から・酔いどれマーチ・チャンス一撃・東京ドーム・ワッショイ・仙台・西日本・沖縄）
- **その他の応援歌**（Go toチャンス・初回テーマ・チャンスメイクテーマ等）
- **球団歌・チームソング**（ファイターズ讃歌・お〜い北海道等）

## ✨ 機能

- 🔍 **選手名・歌詞で検索** — 検索すると該当箇所をハイライト表示し自動スクロール
- 📱 **スマホ最適化** — 球場でもサッと確認できるモバイルファーストデザイン
- 🗂️ **カテゴリタブ** — 選手別 / 共用テーマ / チャンス / 球団歌 で絞り込み
- 📖 **アコーディオン表示** — 歌詞をタップで開閉

## 🚀 使い方

### ローカルで見る
```
index.html をブラウザで直接開くだけ（サーバー不要）
```

### GitHub Pages で公開する
1. このリポジトリを GitHub にプッシュ
2. Settings → Pages → Source: `main` branch, `/ (root)` を選択
3. 数分後に `https://<username>.github.io/<repo>/` でアクセス可能

## 📝 情報更新について

`index.html` 内の各 `.song-card` の `data-name` と `.lyrics-text` を編集することで歌詞・選手を更新できます。

---

> ※歌詞情報は公式サイト（fighters.co.jp）および応援団（全国闘将会）の情報をもとにまとめています。
